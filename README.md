# CNS_Honors_Thesis

This is all the code I wrote for my thesis of which I submitted in partial fulfilment of my B.S. in Neural Science at NYU in 2026. 

Broadly speaking, this code alternates between python and R. In python, I mostly prepare data to be visualized and train the ML model using scikit learn. In R, I conduct the most of the gene ontology analyses using notable packages from Bioconductor. 

I wrote a considerable amount of code (especially the functions), by myself, but I also resourced the web for a considerable amount. In the resources used file, you can find a list of resoruces I kept track of throughout the process that I pulled upon at different points. Some things turned out to be fruitful, while others (e.g., upsampling your data), did not. 

In any case, thanks for reading and I hope you enjoy.
